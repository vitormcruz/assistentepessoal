# harness-arreio

## Propósito

Comunicar o que é um Harness usando a analogia do arreio de cavalo: o LLM
é a força bruta, o harness é o conjunto de controles que o torna funcional
e seguro.

## Mensagens

- O LLM sozinho tem potência mas não tem direção, freio nem carga definida
- O harness é composto por 5 peças:
  - AGENTS.md — a rédea que direciona o comportamento
  - Skills — a bagagem de conhecimento carregada sob demanda
  - MCP — as conexões com o mundo externo
  - Hooks — o freio que bloqueia ações perigosas
  - Sub-agentes — cavalos especializados que dividem a carga
- Cada peça tem uma função clara e complementar
- O harness é o que diferencia um LLM puro de um agente funcional

## Decisões

- Usar a analogia do arreio de cavalo como fio condutor
- Cada componente do harness deve ser identificável e ter sua função
  explicitada
