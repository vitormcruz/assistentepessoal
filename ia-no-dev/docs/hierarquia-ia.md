# hierarquia-ia

## Propósito

Comunicar a relação Prompt ⊂ Contexto ⊂ Harness — cada camada contém a
anterior, e o foco da engenharia de IA sobe de nível conforme o tempo.

## Mensagens

- Prompt vive dentro de Contexto — o prompt é uma fração do que o modelo vê
- Contexto vive dentro de Harness — o contexto é gerenciado pelo ambiente
- Harness é a camada mais externa e mais impactante
- A evolução da engenharia de IA foi: Prompt (2022-24) → Contexto (2025) →
  Harness (2026)
- Resolver problemas na camada certa é mais eficaz que otimizar a camada
  errada

## Decisões

- A relação de contenção (⊂) deve ser o elemento visual central
