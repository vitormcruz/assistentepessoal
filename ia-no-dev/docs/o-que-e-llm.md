# o-que-e-llm

## Propósito

Comunicar o que é um Large Language Model de forma simples e visual: um
motor de predição de texto que recebe entrada, processa e gera saída.

## Mensagens

- Entra texto (prompt), sai texto (completion)
- O funcionamento é predição probabilística de próximo token — não é
  raciocínio consciente
- Foi treinado sobre um conjunto massivo de texto e código
- É uma rede neural, não um banco de dados nem uma consciência

## Decisões

- Deve ser simples e visual, substituindo a tabela de "Confusão comum vs Realidade"
- Deve comunicar os 4 pontos acima sem densidade de texto
